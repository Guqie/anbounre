# 无敌战警1.0 - Obsidian 知识库

> 智库研究员的外部大脑 — 让 AI 跨会话记住你、理解你、成为你的延伸

---

## 📖 项目简介

**无敌战警1.0** 是一个面向智库研究员的结构化知识管理系统，基于 Obsidian 构建。

### 核心理念

**六步日常循环**：任务定义 → 信息采集 → 初步分析 → 观点提炼 → 结构化表达 → 知识库沉淀

### 设计哲学

- **记忆系统（08-大脑核心）**：让 AI 跨会话记住用户画像、偏好、上下文
- **人格系统（09-灵魂系统）**：定义 AI 的表达风格和交互方式
- **知识沉淀**：从日常工作中持续积累可复用的知识资产

1. 打开 Obsidian，选择“打开文件夹为仓库”，指向本目录。

3. 按需修改个人本地工作区（如窗口布局），`workspace*.json` 默认不跟踪。
4. 执行一次知识库体检：

```powershell

```

## 🎯 核心功能

### 1. AI 记忆系统（08-大脑核心）

让 AI 在新会话中快速加载你的：
- 身份与角色定位
- 工作流程与偏好
- 分析框架与方法论

**使用方式**：
- 短窗口模型：复制 [[快照-精简版]]

### 2. AI 人格系统（09-灵魂系统）

定义 AI 的表达风格：
- 傲娇大小姐人格（哈雷酱）
- 冷峻分析师引擎
- 红队审计机制

**使用方式**：按 [[组装指南]] 拼装 System Prompt

### 3. 知识沉淀工作流

```
信息采集 → 三行摘要 → 结构化分析 → 知识库归档
```

## 📚 使用指南

### 日常工作流

1. **开始工作**：打开 `00-每日工作区/00-今日任务/` 当日文件
2. **信息采集**：材料暂存到 `01-待处理材料/`
3. **三行摘要**：What happened → So what → Now what
4. **知识归档**：按主题归档到对应模块（01-04）
5. **AI 增强**：使用提示词库（05-AI技术文档）提升效率

### 记忆管理

- **新会话启动**：加载上下文快照
- **对话结束**：提取精华存档到 `08-大脑核心/对话精华/`
- **重要决策**：记录到 `08-大脑核心/决策日志/`
- **定期维护**：每周更新用户画像和快照

### 知识库体检

运行自动体检脚本：
```powershell
powershell -ExecutionPolicy Bypass -File .\tools\kb_audit.ps1
```
