# AI技能进阶路线

> 目标岗位：大模型应用开发工程师（15-20K）
> 关联：[[职业规划-AI转型路径]] | [[简历-AI方向]]
> 知识沉淀：[[05-AI技术文档/]]

---

## 定位说明

**你已经具备的能力（不需要从零学）：**
- ✅ PyTorch + CNN/LSTM/MLP
- ✅ BERT微调（NER F1:97.4%）
- ✅ LangChain + RAG Pipeline
- ✅ Neo4j知识图谱（30万三元组）
- ✅ Streamlit前端
- ✅ 多Agent协作系统
- ✅ Python数据工程（pandas/ETL）

**你缺的不是技术广度，而是两个硬伤：**
- ❌ 没有GitHub公开项目（简历筛选直接减分）
- ❌ 没有工程化部署能力（Docker/FastAPI是JD标配）

本路线聚焦：**补齐两个硬伤 + 把已有能力打磨到面试水平**

---

## 学习原则

1. **GitHub先行** — 第一周就开始整理代码上传，这是简历的"门面"
2. **复习优先** — 先把已有项目吃透，能讲清楚每个技术选型的why
3. **刷题并行** — LeetCode每天1-2题，面试硬通货
4. **输出驱动** — 每完成一个模块写笔记沉淀到 [[05-AI技术文档/]]

---

## P0 — 拿到面试的硬门槛（第1-3周）

### 1. GitHub Portfolio 整理（第1周，最优先！）

- [ ] 创建GitHub账号，完善个人Profile（头像、Bio写"AI应用开发"）
- [ ] 上传KG+LLM心理咨询QA系统：清理代码、写README（架构图+运行说明+效果截图）
- [ ] 上传RAG+多Agent简报系统：同上
- [ ] 每个项目README必须包含：项目简介、架构图、技术栈、运行方式、效果展示
- [ ] 输出：2个可展示的GitHub项目仓库

### 2. 项目复盘与面试话术（第1-2周，与GitHub同步进行）

- [ ] KG+LLM QA系统：画架构图、梳理每个模块的技术选型理由
- [ ] RAG+Agent简报系统：同上，重点准备"为什么这样设计"的回答
- [ ] 每个项目准备3分钟版和10分钟版讲解话术
- [ ] 输出：每个项目一份《技术复盘文档》沉淀到 [[05-AI技术文档/]]

### 3. LeetCode刷题启动（第1-3周持续）

- [ ] 第1周：数组、字符串、哈希表（双指针、滑动窗口）
- [ ] 第2周：链表、栈/队列、树（BFS/DFS、层序遍历）
- [ ] 第3周：动态规划基础（背包、最长子序列）
- [ ] 目标：每天1-2题，3周内完成50题
- [ ] 重点：Python解法，注意时间/空间复杂度分析
- [ ] 推荐题单：LeetCode Hot 100 精选

---

## P1 — 工程化补强 + 八股文（第3-6周）

### 4. FastAPI服务化（第3-4周）

- [ ] FastAPI基础：路由、Pydantic模型、依赖注入、中间件
- [ ] 将毕业设计QA系统封装为REST API（至少3个接口：问答、KG查询、健康检查）
- [ ] Swagger自动生成接口文档
- [ ] 异步处理：大模型调用的async/await
- [ ] 输出：一个可通过API调用的AI问答服务（上传GitHub）

### 5. Docker与部署（第5周）

- [ ] Docker基础：Dockerfile编写、镜像构建、容器管理
- [ ] docker-compose编排多服务（Neo4j + FastAPI + Streamlit）
- [ ] 了解基本的云部署流程（阿里云ECS / 腾讯云轻量）
- [ ] 输出：项目Docker化，README里加上 `docker-compose up` 一键启动说明

### 6. 八股文 + 向量数据库（第6周）

- [ ] Transformer架构（自注意力、多头注意力、位置编码、KV Cache）
- [ ] BERT原理（MLM、NSP、微调 vs Feature-based、与GPT区别）
- [ ] RAG全流程（切分策略、Embedding选型、向量检索、Rerank）
- [ ] 向量数据库：Milvus / Chroma / Weaviate 选型对比
- [ ] 向量索引原理（HNSW、IVF）
- [ ] Embedding模型选型（BGE、M3E、text2vec）
- [ ] 优化RAG检索效果：Rerank策略、混合检索（稀疏+稠密）
- [ ] 输出：面试速查笔记 + 《向量数据库实践笔记》沉淀到 [[05-AI技术文档/]]

---

## P2 — 竞争力加分（第7-10周，入职前后均可）

### 7. 模型微调实战（第7-8周）

- [ ] LoRA / QLoRA 微调原理与实践
- [ ] 用自有数据微调一个垂直领域小模型（推荐：用心理咨询数据微调Qwen）
- [ ] 评估指标设计与效果对比（微调前 vs 微调后）
- [ ] 输出：微调实战笔记 + GitHub代码

### 8. AI Agent进阶（第9-10周）

- [ ] LangGraph / CrewAI 深入学习
- [ ] 复杂Agent工作流设计（ReAct、Plan-and-Execute、反思循环）
- [ ] Tool Use / Function Calling 最佳实践
- [ ] 输出：一个可演示的Agent项目（上传GitHub）
