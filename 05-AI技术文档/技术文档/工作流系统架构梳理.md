## 一、系统架构全景图

当前有三层系统，但彼此**未打通**：

```
┌──────────────────────────────────────────────────────┐
│                     日常工作流                         │
│  信息采集 → 筛选分析 → 三行摘要 → 报告产出 → 知识沉淀    │
└──────┬──────────────┬──────────────────┬──────────────┘
       ↓              ↓                  ↓
  ┌─────────┐   ┌────────────┐   ┌──────────────┐
  │ Layer 1  │   │  Layer 2   │   │   Layer 3    │
  │ 信息采集  │   │  处理分析   │   │  知识管理     │
  │          │   │            │   │              │
  │ news-    │   │ 无敌战警-   │   │ 无敌战警1.0   │
  │ workflow │   │ engine     │   │ (Obsidian)   │
  └─────────┘   └────────────┘   └──────────────┘
   C:/Users/      D:/桌面/          D:/桌面/
   guqi/          无敌战警-engine/   无敌战警1.0/
```

### 核心断裂点

- news-workflow 采集的数据 → ❌ → engine 的 SQLite 队列
- engine 处理的结果 → ❌ → Obsidian 知识库（部分可用但没自动化）

## 三、Layer 2：处理分析层（无敌战警-engine）

**位置：** `D:/桌面/无敌战警-engine/`

| 组件 | 功能 | 状态 |
|------|------|------|
| AIAnalyzer | 三行摘要/洞察提炼/证据链/趋势分析/弱信号/PIDST分类 | ✅ 完整 |
| KnowledgeManager | 归档/搜索/标签/关联发现/索引管理 | ✅ 完整 |
| DatabaseManager | SQLite材料队列（导入→处理→归档） | ✅ 完整 |
| TaskScheduler | 每日模板/复盘/周总结/进度追踪 | ✅ 完整 |
| Agent-全知之眼 | 情报分析/三行摘要/趋势研判 | ✅ 完整 |
| Agent-真理之锚 | 逻辑审计/架构评审 | ✅ 完整 |
| Agent-智学导师 | 苏格拉底式教学/考研辅助 | ✅ 完整 |
| Flask RSS网关 | RSS聚合+LLM摘要+关键词过滤 | ✅ 可用但源太少 |

### CLI 命令（main.py）

```bash
python main.py daily          # 生成每日任务模板
python main.py analyze        # 分析材料→三行摘要
python main.py insight        # 提炼洞察
python main.py evidence       # 构建证据链
python main.py trend          # 趋势分析
python main.py archive        # 归档知识
python main.py review         # 生成复盘模板
python main.py weekly         # 周总结
python main.py search         # 搜索知识库
python main.py stats          # 统计信息
python main.py import-gov     # 导入政府库材料
python main.py import-ind     # 导入行业库材料
python main.py process        # 批量处理队列→三行摘要→归档
```

### 关键能力：`process` 命令

这是打通 Layer 1 → Layer 3 的关键桥梁：
- 从 SQLite 队列取待处理材料
- 调用 AI 生成三行摘要
- 自动 PIDST 分类
- 归档到对应知识库目录

## 六、关于 MSN/微软新闻 RSS

**结论：微软官方不提供中文新闻RSS源。**

- Microsoft Start（原MSN新闻）是聚合平台，不对外提供RSS订阅
- RSSHub 社区曾请求支持MSN新闻（Issue #17582），已被关闭
- Bing News Search API 可搜新闻，但是付费API
