# MCP 信息采集能力测试报告

> 测试日期：2026-02-23
> 测试工具：exa (web_search_exa) / open-websearch (DuckDuckGo)
> 测试目的：评估 MCP 搜索工具对各负责领域的中文新闻采集能力

| 人形机器人 | ⭐⭐⭐⭐⭐ | ⭐⭐（仅2条知乎） | exa |

| 热点风口/战略新兴 | ⭐⭐⭐⭐⭐ | 未测 | exa |

**核心结论：exa 是中文产业新闻搜索的最佳 MCP 工具，返回结果权威、时效性强、覆盖面广。open-websearch 对中文新闻搜索能力较弱，仅适合作为补充。**

| 适用场景 | **主力搜索工具** | 英文搜索/补充验证 |

## 四、优化建议

### 立即可用的采集策略

1. **主力工具：exa** — 所有领域均表现优异，直接用于日常信息采集
2. **搜索词模板：** `{领域关键词} 产业 政策 发展 2026` 效果最佳
3. **每日采集建议：** 9个领域 × 1次搜索 × 5条/次 = 45条/日，筛选后保留10-15条

### 搜索词优化建议

| 领域 | 推荐搜索词 |
|------|-----------|
| 智能驾驶 | `智能驾驶 自动驾驶 产业 政策 2026` |
| 人形机器人 | `人形机器人 具身智能 产业 发展 2026` |
| 低空经济 | `低空经济 eVTOL 无人机 政策 2026` |
| 商业航天 | `商业航天 卫星 火箭 产业 2026` |
| 氢能发展 | `氢能 燃料电池 绿氢 产业 2026` |
| 人工智能 | `人工智能 大模型 AI应用 政策 2026` |
| 低碳经济 | `低碳经济 碳中和 碳交易 绿色转型 2026` |
| 热点风口 | `战略新兴产业 未来产业 科技趋势 2026` |

### 后续开发方向

1. **自动化脚本：** 将 exa 搜索集成到 news-workflow，每日自动采集9个领域
2. **去重机制：** 基于URL和标题相似度去重
3. **质量评分：** 按信源权威性自动评分（部委>央媒>财经媒体>门户>自媒体）
4. **三行摘要：** 采集后自动调用 engine 的 AI 分析生成三行摘要
5. **RSS 补充：** 安装 `@missionsquad/mcp-rss` 订阅固定信源，与 exa 搜索互补
