# aio-coding-hub 使用指南

> All In One 本地 AI 工具 — CC/CX/Gemini 多供应商管理 + 代理熔断 + 用量统计 + 模型验证

**来源**: [linux.do 社区分享](https://linux.do/t/topic/1479596)

---

## 一、aio-coding-hub 是什么

### 核心定位

**aio-coding-hub** — 一个 All In One 的本地 AI 工具

- 专为使用中转站的 CC/CX/Gemini 用户设计
- 支持 Win/Mac，打开即用，无需服务器或 Docker
- 集成供应商管理、代理熔断、用量统计、模型验证等功能
- 融合了 cc-switch、code-switch-r、claude-code-hub 的优点

### 设计理念

作者在使用站内多个 CC/CX/Gemini 管理软件后，发现各有痛点：

| 工具 | 优点 | 痛点 |
|------|------|------|
| **cc-switch** | 简约直观 | 缺少精细化日志和统计 |
| **code-switch-r** | 代理模式强大，SLA 约等于 100% | 功能相对单一 |
| **claude-code-hub** | 功能无敌 | 必须有服务器或 Docker，配置复杂 |

**aio-coding-hub 的解决方案**: 把这些工具的优点全部"缝合"，下载即用，并增加自己的创新想法。

## 二、核心功能

### 1. CC/CX/Gemini 供应商管理

**多供应商管理**:
- 支持添加多个中转站供应商
- 倍率设置（自定义价格倍率）
- 供应商启用/禁用控制

**模板顺序** (核心特色):
- 按场景切换供应商组合（如"工作"/"生活"）
- 一键切换不同供应商模板
- 适合多场景使用需求

### 2. 代理模式

**自动熔断机制**:
- 中转站天生不稳定，代理模式提供高可用性
- 供应商挂了自动拉黑降级
- SLA 约等于 100%

**多 BaseUrl 支持**:
- 支持为单个供应商配置多个 BaseUrl
- 按排序或 Ping 耗时发起请求
- 提升请求成功率

### 3. 用量统计

**预计花费**:
- 实时统计 API 调用费用
- 支持自定义倍率计算
- 花费趋势可视化

**热力图**:
- 使用时间热力图
- 快速识别高频使用时段

**性能指标**:
- 请求耗时统计
- 首字时间（TTFB）
- 缓存命中率

### 4. 模型验证 (核心创新)

**背景**: 短期内出现多起"富可敌国 CC 掺假"事件（1元1刀甚至1.5元1刀，结果给混逆向）

**验证机制**:
- 检测 CC 是否符合规范
- 识别假模型和混逆向
- 帮助用户避免被欺骗

**重要说明**:
- 使用中转站理论上会产生中间人
- 真想做也能实现 100% 对齐官方
- 验证通过的不一定 100% 是真的
- 但能揪出一些浑水摸鱼的

### 5. Skill/MCP/提示词管理

- 集成 Skill 管理
- MCP 工具配置
- 提示词库管理
- 统一管理 AI 工作流资源

## 四、安装与使用

### 安装

```bash
# 从 GitHub 下载最新版本
# 支持 Win/Mac，打开即用
```

**GitHub 仓库**: https://github.com/dyndynjyxa/aio-coding-hub

### 基本使用流程

```bash
# 1. 下载并解压
# 2. 打开应用
# 3. 添加供应商
# 4. 配置代理模式（可选）
# 5. 开始使用
```

### 供应商配置

1. **添加供应商**: 输入 API Key、BaseUrl、倍率
2. **创建模板**: 按场景组合供应商（工作/生活）
3. **启用代理模式**: 配置多 BaseUrl 和熔断策略
4. **模型验证**: 点击验证按钮检测供应商真实性

## 八、使用建议

### 适合人群

✅ 使用中转站的 CC/CX/Gemini 用户
✅ 需要多供应商管理的用户
✅ 关注用量统计和成本控制的用户
✅ 担心供应商掺假的用户
✅ 需要高可用性的用户

### 不适合人群

❌ 只使用官方 API 的用户
❌ 不关心用量统计的用户
❌ 需要企业级功能的用户（推荐 claude-code-hub）

### 核心价值

1. **降低风险**: 模型验证功能帮助识别假供应商
2. **提升可用性**: 代理模式 + 自动熔断保证 SLA
3. **精细管理**: 详细的用量统计和成本预测
4. **场景切换**: 模板系统支持多场景快速切换
5. **开箱即用**: 无需复杂配置，下载即用
