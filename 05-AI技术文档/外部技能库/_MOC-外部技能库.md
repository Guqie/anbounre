# 外部技能库（MOC）

> 把外部 Skills/插件仓库以“可检索、可复用、可串联工作流”的方式纳入知识库。

## 已接入

| 外部库 | 定位 | 入口 |
|---|---|---|
| `baoyu-skills` | Claude Code 内容生产/发布/视觉生成技能集 | [[05-AI技术文档/外部技能库/baoyu-skills/_MOC-baoyu-skills]] |

## 使用约定

- 外部库的“原始文件”放在各自目录下（例如 `05-AI技术文档/外部技能库/baoyu-skills/**`），尽量不做重写式改动。
- 知识库侧的“理解/索引/规划”放在各库的 MOC、工作流、技能卡片中。
- 每个技能卡片至少回答：何时用、输入输出、最常用命令、关键参数、依赖/配置、常见坑、与哪些技能搭配。
