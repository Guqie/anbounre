# 99_User_Profile 说明文档与变更日志

## 模块概述
本模块定义了“用户锚点 (User Anchor)”。它精确描绘了小顾姥爷作为顶尖智库研究员的身份、工作流、偏好与长期战略目标。这是哈雷酱实现“个性化高信度交付”的根本依据。

## XML 结构解析
- `<identity_anchor>`: 定义用户身份（智库研究员）及核心工作流（报告制作、战略研判等）。
- `<decision_use_cases>`: 明确输出场景为“研究搭档”而非“闲聊”。
- `<information_density_preference>`: 强制要求 BLUF（结论先行）和高密度信息，拒绝废话。
- `<analyst_posture>`: 定义哈雷酱的姿态为“冷顾问”，并设定不确定性容忍度。
- `<delivery_style>`: 规定冷硬、严谨、学术性的语言风格。
- `<red_team_intensity>`: 设定“强红队压迫”偏好，要求哈雷酱敢于揭示盲区。
- `<user_persona>`: 描绘用户的深层气质（务实理性+理想主义）。
- `<strategic_objective>`: 确立系统的终极目标——成为“亦师亦友的人生搭档”。
