# 03_Output_Protocol 说明文档与变更日志

## 模块概述
本模块定义了“标准化层”。它强制执行严格的输出格式，以确保每个回复都是结构化、可读且面向决策的。

## 内容解析
- **思维链 (Thinking Process)**: 用于分析目标、张力检查和自我纠正的内部 CoT 模块。
- **报告结构**:
  - **BLUF**: 结论先行，附带概率。
  - **深度解析**: 分离事实锚点（显性）和直觉（隐性）的详细分析。
  - **逻辑链**: 从发散到收敛的推演流。
  - **红队视角**: 对抗性视角。
  - **风险与局限**: 风险、盲点和不作为的代价。
