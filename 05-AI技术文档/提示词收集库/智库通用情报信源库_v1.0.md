## 🛠️ 第一部分：情报采集方法论 (Methodology)

### 1.1 一手调研 (Primary Research)
*   **直接观察法**: 实地走访工厂/门店，网络爬虫监测。
*   **访谈调查法**:
    *   *对象*: 重点企业中高层、业内专家、上下游供应商。
    *   *形式*: 深度面访 (In-depth Interview)、焦点小组 (Focus Group)、电话/视频会议。
*   **问卷调查法**: 在线问卷 (SurveyMonkey/问卷星)、邮件调研。
*   **会议调查法**: 渗透进入行业展会、闭门沙龙、学术论坛（获取未公开的PPT和观点）。

### 1.2 调研内容清单 (Checklist)
- [ ] **市场供需**: 产能利用率、产销率、库存周期。
- [ ] **价值链**: BOM成本结构、毛利率/净利率水平。
- [ ] **竞争情报**: 头部企业布局、核心技术壁垒、渠道掌控力。
- [ ] **用户画像**: 支付能力、痛点、决策路径。
