## 数据流架构

```mermaid
sequenceDiagram
    participant User as 用户
    participant Agent as AI Agent
    participant MPM as MemoryPersonaManager
    participant MIM as MemoryIndexManager
    participant PL as PersonaLoader
    participant KB as 知识库

    User->>Agent: 初始化 Agent
    Agent->>MPM: load_full_context(recipe)

    MPM->>MIM: get_user_profile()
    MIM->>KB: 读取 08-大脑核心/用户画像.md
    KB-->>MIM: 用户画像内容
    MIM-->>MPM: 返回用户画像

    MPM->>MIM: get_preferences()
    MIM->>KB: 读取偏好设定文件
    KB-->>MIM: 偏好设定内容
    MIM-->>MPM: 返回偏好设定

    MPM->>PL: assemble_prompt(recipe)
    PL->>KB: 读取 09-灵魂系统/模块文件
    KB-->>PL: 人格模块内容
    PL-->>MPM: 返回 System Prompt

    MPM->>MIM: get_context_snapshot()
    MIM->>KB: 读取上下文快照
    KB-->>MIM: 快照内容
    MIM-->>MPM: 返回快照

    MPM-->>Agent: 返回完整上下文
    Agent-->>User: Agent 就绪（带人格+记忆）
```

## 使用流程

### 1. 重建记忆索引

```bash
cd D:/桌面/无敌战警-engine/
python main.py rebuild-memory-index
```

### 2. Agent 自动加载

```python
from workflow_automation.agents.intelligence import IntelligenceAgent
from workflow_automation.utils import load_config

# 加载配置
config = load_config()

# 初始化 Agent（自动加载记忆和人格）
agent = IntelligenceAgent('test', config)

# 获取 System Prompt（包含哈雷酱人格）
prompt = agent.get_system_prompt()

# 获取用户画像
profile = agent.get_user_profile()

# 获取偏好设定
preferences = agent.get_preferences()
```

### 3. 切换人格配方

```python
# 在 config.yaml 中修改
memory_persona:
  default_recipe: "full"  # 或 "compact" 或 "analysis_only"
```
