# 🧠 大脑核心 (Memory System MOC)

> 这里存储的不是知识，而是"你是谁"——让任何大模型在新会话中秒懂你。

---

## 核心文件

| 文件 | 用途 |
|------|------|
| [[用户画像]] | 你的身份、角色、工作流、偏好的完整描述 |
| [[输出风格偏好]] | AI 输出的格式、语调、密度偏好 |
| [[分析框架偏好]] | 你常用的分析方法论和思维框架 |

## 记忆存储

| 目录 | 用途 |
|------|------|
| [[项目记忆/]] | 每个项目的上下文记忆（目标、进展、决策） |
| [[对话精华/]] | 高价值对话片段的结构化存档 |
| [[决策日志/]] | 重要决策的记录与复盘 |

## 上下文快照

| 文件 | 适用场景 |
|------|---------|
| [[快照-完整版]] | 长上下文模型（Claude/GPT-4）— 完整记忆注入 |
| [[快照-精简版]] | 短窗口模型（DeepSeek/本地模型）— 核心信息压缩 |

---

## 使用流程

1. **新会话启动** → 复制对应快照内容，粘贴到 System Prompt 或首条消息
2. **会话结束** → 有价值的内容用[[对话精华模板]]存档到 `对话精华/`
3. **重要决策** → 用[[决策日志模板]]记录到 `决策日志/`
4. **定期维护** → 每周更新[[用户画像]]和快照文件
