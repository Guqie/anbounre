### ContextBlock 数据结构

```
┌─────────────────────────────────────────────────────────────────┐
│                        ContextBlock (上下文块)                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ┌─────────────┐   intent: ┌────────────────┐                  │
│  │    id:      │──▶        │ directive       │ (指令,必须落system)│
│  │ "user_profile"          └────────────────┘                  │
│  └─────────────┘          intent: ┌────────────────┐          │
│                              │ reference      │ (参考,可降级user)│
│  kind: "user_profile"        └────────────────┘                  │
│                                                                 │
│  priority: 2          # 优先级 (1=最高)                         │
│  min_tokens: 200       # 最小保证 (不够则digest)                 │
│  cap_tokens: 1000     # 最大上限 (超过则截断)                    │
│                                                                 │
│  compress: "kv_digest" # 压缩策略                              │
│  persistence: "permanent"  # 持久化级别                         │
│                                                                 │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ content: "用户画像内容..."                                │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│  tokens: 450           # 当前token数                            │
│  compressed: false     # 是否已压缩                             │
│  compression_ratio: 1.0  # 压缩比                              │
│                                                                 │
│  meta: {}              # 元数据                                 │
│                                                                 │

```
┌─────────────────────────────────────────────────────────────────┐

    │  │  │ system_prompt      │  │ context_blocks     │  │  │

    │  │  └────────────────────┘  └────────────────────┘  │  │
    │  └────────────────────────────────────────────────────┘  │
    │  ┌────────────────────────────────────────────────────┐  │
    │  │              reserve: 1500 tokens                   │  │
    │  │         (用户问题 + 模型响应的硬保留)                 │  │
    │  └────────────────────────────────────────────────────┘  │
    └──────────────────────────────────────────────────────────┘

    ┌─────────────────────────────────────────────────────────┐
    │                     Budget 计算公式                        │
    │                                                         │
    │   total = reserve + available                           │
    │   available = system_prompt + context_blocks            │
    │                                                         │
    │   约束: injected_tokens ≤ (total - reserve)             │
    │                                                         │
