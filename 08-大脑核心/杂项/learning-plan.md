# 学习计划量化方案

## 📊 时间资源分析

**工作日可用时间：**
- 晚上个人时间：19:30-01:00（5.5小时）
- 建议学习时间：19:30-23:30（4小时）
- 休息娱乐：23:30-01:00（1.5小时）

**周末可用时间：**
- 每天：8-10小时
- 每周总计：16-20小时

**每周总学习时间：**
- 工作日：4小时 × 5天 = 20小时
- 周末：8小时 × 2天 = 16小时
- **总计：36小时/周**

## 🎯 短期目标量化（3-6个月）

### 1. 英语学习

**目标：**
- 词汇量：从当前水平提升到6000词（考研要求）
- 语法：掌握考研核心语法

**每日计划：**
- 背单词：30分钟（50-80个新词 + 复习）
- 语法练习：20分钟（1-2个语法点）
- 阅读练习：20分钟（1篇文章）
- **每日总计：70分钟（约1小时）**

**量化指标：**
- 每天：50个新单词 + 复习100个旧单词
- 每周：完成1套语法练习题
- 每月：词汇量增加600-800词

**工具：**
- 背单词：Anki、墨墨背单词
- 语法：考研语法书
- 阅读：考研真题阅读

### 2. 编程能力提升

**目标：**
- Python：掌握数据结构、算法、Web开发
- 前端：HTML/CSS/JavaScript基础
- 后端：Flask/Django框架

**每日计划：**
- 编程练习：1小时（算法题 + 项目实践）
- 视频学习：30分钟（系统学习课程）
- 项目开发：30分钟（实际项目）
- **每日总计：2小时**

**量化指标：**
- 每天：1-2道算法题（LeetCode/牛客网）
- 每周：完成1个小项目或功能模块
- 每月：完成1个完整项目

**学习路径：**
- 第1-2月：Python基础 + 数据结构与算法
- 第3-4月：前端基础（HTML/CSS/JS）
- 第5-6月：后端框架（Flask/Django）

**工具：**
- 算法练习：LeetCode、牛客网
- 视频课程：B站、慕课网
- 项目托管：GitHub

## 🎓 中期目标量化（6-12个月）

### 考研准备（人工智能专业）

**考试科目：**
1. 政治（100分）
2. 英语一（100分）
3. 数学一/数学二（150分）
4. 专业课（150分）- 通常是数据结构、计算机组成原理等

**时间规划：**
- 前6个月：基础积累（英语+编程+数学基础）
- 后6个月：系统备考（全科目强化）

**每日计划（后6个月）：**
- 英语：1小时
- 数学：1.5小时
- 专业课：1小时
- 政治：30分钟（后3个月加强）
- **每日总计：4小时**

**量化指标：**
- 英语：词汇量6000+，真题正确率70%+
- 数学：完成全部考研数学真题
- 专业课：完成目标院校历年真题
- 政治：选择题正确率80%+

**里程碑：**
- 6个月：完成基础积累，开始系统备考
- 9个月：完成第一轮复习
- 11个月：完成第二轮强化
- 12个月：冲刺模拟，参加考试

## 📈 进度追踪方式

### 每日记录
- 在 `memory/YYYY-MM-DD.md` 中记录每日学习内容
- 记录格式：
  ```
  ## 学习记录
  - 英语：背单词50个，语法练习1套
  - 编程：完成2道算法题，项目进度XX%
  - 总结：今天的收获和问题
  ```

### 每周回顾
- 每周日晚上进行周回顾
- 检查本周目标完成情况
- 调整下周计划

### 每月总结
- 更新 `memory/learning-progress.md`
- 记录里程碑达成情况
- 分析问题和改进方向

### 量化指标追踪
- 英语词汇量：使用Anki统计
- 算法题数：LeetCode/牛客网统计
- 项目进度：GitHub commit记录
- 学习时长：使用番茄钟/时间追踪工具

## 💡 执行建议

### 1. 优先级原则
- 考研是中期核心目标，优先保证
- 英语和编程是基础，每天必须完成
- 灵活调整，但不能连续中断

### 2. 防止倦怠
- 每周至少1天完全休息
- 学习1小时休息10分钟
- 保持兴趣，选择感兴趣的项目

### 3. 利用碎片时间
- 通勤时间：听英语、背单词
- 午休时间：复习笔记
- 睡前时间：轻松阅读

### 4. 寻求支持
- 加入学习社群
- 找学习伙伴互相监督
- 定期与我（哈雷酱）回顾进度
