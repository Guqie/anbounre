# 智库研究员每日工作检查清单

## 📋 快速执行版（可打印贴墙）

### ⏰ 开始工作前（3分钟）
- [ ] 打开今日任务文件
- [ ] 明确今日1-3个核心任务
- [ ] 确定任务类型分类
- [ ] 设定今日最小交付物

---

### 🔍 信息采集阶段（45-90分钟）
- [ ] **P政策** - 检索部委/省级/市级政策
- [ ] **I产业** - 收集医疗健康/人才教育动态
- [ ] **D数据** - 获取官方统计和行业数据
- [ ] **S信号** - 监测舆情和弱信号
- [ ] **T技术** - 跟踪AI+医疗/教育技术

### 📝 信息处理
- [ ] 信息暂存到待处理材料文件
- [ ] 完成三行摘要处理
- [ ] 按重要程度标记（🔴🟡🟢）
- [ ] 识别信息关联性

---

### 🧠 分析思考阶段（30分钟）
- [ ] 信息归类和分级
- [ ] 识别趋势和模式
- [ ] 发现弱信号
- [ ] 标记深入研究问题

---

### 💡 观点提炼阶段（15分钟）
- [ ] 提炼至少3个核心洞察
- [ ] 每个洞察都有证据支撑
- [ ] 完成洞察模板填写
- [ ] 评估观点的前瞻性

---

### 📊 结构化输出阶段（30-60分钟）

#### 政府库/行业库更新
- [ ] 完成每日更新摘要
- [ ] 更新政策动态
- [ ] 更新行业动态
- [ ] 标记风险提示

#### 决策支持研究
- [ ] 构建问题树
- [ ] 收集证据链（F-D-P-T-S）
- [ ] 完成一页纸简报
- [ ] 提出趋势判断

#### 专题研究推进
- [ ] 选择今日推进项
- [ ] 补充证据链
- [ ] 新增观点（至少1条）
- [ ] 更新专题文件

---

### 📚 知识沉淀阶段（15-20分钟）
- [ ] 三行摘要归档到各专业库
- [ ] 弱信号更新到信号库
- [ ] 观点更新到洞察库
- [ ] 方法论更新到方法库
- [ ] 专题内容归档

---

### 🔄 复盘总结阶段（5-8分钟）
- [ ] 记录今天3个做得好的点
- [ ] 总结遇到的问题
- [ ] 确定明日优化点
- [ ] 设定明日重点任务
