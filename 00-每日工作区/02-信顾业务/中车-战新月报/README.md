## 客户关注领域

### 清洁能源装备
- 风电装备（整机、叶片、发电机、塔筒、齿轮箱）
- 光伏装备（**仅关注逆变器**）
- 储能装备（锂电储能、飞轮储能、超级电容）
- 氢能装备（制氢、运输、加氢、动力、储氢）
- 其他（绿色燃料发动机、柴电机组、输变电、电驱系统）

### 数智及未来产业
- 智能制造系统
- 工业机器人（本体、集成、海工）
- 工业软件（研发设计、生产制造、运维服务）
- 功率半导体（**仅功率半导体 + 传感器**）
- 新材料（复合材料、气体分离、膜产品）
- 海工装备（船舶发动机、船用增压器）
- 节能环保（环保材料、污水处理）
- 低空装备（无人机、动力叶片）

## 编辑注意事项

1. 删除人名、专家名字（删除"专家、认为、分析、表示、介绍"等表述）
2. 资讯第一句和标题意思重复时不要加粗
3. 标题序号后加一个空格（如 `1. 标题`）
4. 不要有中车自身新闻
5. 数字、英文字体使用 Times
